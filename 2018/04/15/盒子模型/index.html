盒子模型

上面是一张盒子模型图，一个盒子模型包括 content（实际内容）、border（边框）、padding（内边距）和 margin（外边距）。

盒子模型有两种，分别是W3C盒子模型和IE盒子模型 。

1. W3C盒子模型

W3C定义的盒子模型包括了margin、border、padding、content ，其中元素的 width = content 的宽度，height = content 的高度。

2. IE盒子模型

IE盒子模型与W3C的盒子模型唯一区别就是元素的宽度，IE盒子模型元素的 width = content + padding + border，而height = content + padding + border 。

在我的理解中，W3C定义盒子模型与IE定义的盒子模型，IE定义的比较合理，因为一个盒子就应该包括 content、padding 和 border ，现实中的盒子也是如此。因此W3C也意识到这个问题，在CSS3中新增了一个新的样式：box-sizing，包含3个属性，分别是 content-box、border-box 和 inherit 。

其中inherit属性不多加解析，就是继承父元素该样式的属性。下面来说说 content-box 和 border-box 。
